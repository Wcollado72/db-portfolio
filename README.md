# Database & SQL Portfolio

This repository contains my personal database and SQL projects.  
The goal is to demonstrate my skills as a future Database Developer / DBA, including relational design, SQL querying, and basic performance tuning.

## Structure

- `mysql/`
  - `project1_clinic/`
    - `schema.sql` – Initial schema for the clinic management project (patients, doctors, appointments, specialties).
    - `seed.sql` – Sample data to populate the tables (to be added).
    - `queries_reports.sql` – Example queries and reports (to be added).
  - `exercises/`
    - Small SQL practice scripts (joins, subqueries, window functions, etc.).

- `postgresql/`
  - `project2_courses/` (planned)
    - Schema and queries for a course enrollment system.

- `azure-sql/` (planned)
  - Labs and scripts related to Azure SQL and preparation for the DP-900 Azure Data Fundamentals exam.

## Technologies

- MySQL Community Edition
- (Planned) PostgreSQL
- (Planned) Azure SQL Database
